# VCU项目
## 软件架构

```
┌───────────────────────┐
│       应用层          │  // 业务逻辑（车辆控制策略、状态机）
├───────────────────────┤
│      服务层           │  // 协议栈、算法库、诊断服务
├───────────────────────┤
│  硬件抽象层（HAL）     │  // STM32外设驱动+第三方传感器驱动
├───────────────────────┤
│        平台层         │  // 微控制器抽象层（基于STM32CubeH7）
└───────────────────────┘
```
## 目录结构
```
app/                               //业务逻辑
common/                     //公共组件
doc/                            //文档
middlewares/               //中间层
platforms/                      //平台层
```
